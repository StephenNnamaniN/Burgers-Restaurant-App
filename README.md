# Courier & Food Delivery App (Android – Kotlin, Jetpack Compose, Firebase)

A modern, production-ready courier and food delivery mobile application built with Kotlin and Jetpack Compose. This app supports multi-role flows (Customer, Supplier, Driver), real-time tracking, product management, and full Firebase integration.

## 🚀 Tech Stack
- **Kotlin**
- **Jetpack Compose**
- **MVVM + Clean Architecture**
- **Firebase Authentication**
- **Cloud Firestore**
- **Firebase Storage**
- **RESTCountries API**

## 🔧 Core Features

### 🔹 Authentication & User Profiles
- Firebase Auth (Google Sign-In + custom profiles)
- Real-time user data sync with Firestore
- Profile photo upload using Photo Picker + Storage
- Country selector with RESTCountries API + dial code population

### 🔹 Admin Product Management
- Add, update, and delete product items
- Upload product images to Firebase Storage
- Real-time UI updates with loading/error states

### 🔹 Courier Features (In Progress)
- Driver, User, and Supplier account roles
- Real-time tracking + geo updates
- Proximity-based job matching
- Mileage/time-based pricing

## 🧱 Architecture
- Clean Architecture with Use Cases & Repository pattern
- Unidirectional data flow
- Modular composables for UI reuse
